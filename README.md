# Mandelbrot Deep Zoom

Static Mandelbrot explorer built with TypeScript, Vite, WebGL2, Web Workers, and Rust WASM reference-orbit math.

## Commands

- `npm install`
- `npm run build:wasm`
- `npm run dev`
- `npm test`
- `npm run build`
- `npm run test:e2e`

The application does not use `SharedArrayBuffer` and does not require cross-origin isolation headers.
